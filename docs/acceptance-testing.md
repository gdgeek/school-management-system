# 功能验收测试计划

## 概述

本文档为学校管理系统的功能验收测试计划，覆盖所有核心业务模块、跨系统集成、认证授权及UI/UX测试。每个测试用例包含用例编号、描述、操作步骤、预期结果和测试状态。

### 测试环境

| 项目 | 说明 |
|------|------|
| 前端地址 | `http://localhost:3002` |
| 后端API地址 | `http://localhost:8081/api` |
| 主系统地址 | `http://localhost:8080` |
| 数据库 | MySQL 8.0+（与主系统共享） |
| 缓存 | Redis 6.0+（与主系统共享） |
| 浏览器 | Chrome 最新版、Firefox 最新版、Safari 最新版 |

### 测试状态说明

| 状态 | 说明 |
|------|------|
| ✅ 通过 | 测试通过，功能正常 |
| ❌ 失败 | 测试失败，存在缺陷 |
| ⬜ 未测试 | 尚未执行测试 |
