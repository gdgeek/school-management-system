# 用户手册

## 系统简介

学校管理系统用于管理学校、班级、教师、学生和学习小组。系统与主平台（XR UGC）集成，支持单点登录，无需单独注册账号。

## 访问系统

### 从主系统跳转

1. 登录主系统（XR UGC 平台）
2. 在顶部导航栏或侧边栏找到"学校管理"链接
3. 点击链接，系统会自动在新标签页打开学校管理系统，并完成自动登录

### 直接访问

如果已有账号，也可以直接访问学校管理系统地址，使用用户名和密码登录。

### 返回主系统

点击页面顶部导航栏中的"返回主系统"链接即可跳转回主平台。

## 学校管理

### 查看学校列表

1. 点击侧边栏"学校管理"
2. 页面显示所有学校的列表，包含学校名称、校长、班级数量等信息
3. 使用顶部搜索框按学校名称搜索
4. 使用底部分页控件翻页

### 创建学校

> 需要系统管理员权限

1. 在学校列表页点击右上角"新建学校"按钮
2. 填写学校信息：
   - **学校名称**（必填）：学校的正式名称
   - **校长**（可选）：从用户列表中选择校长
   - **学校图片**（可选）：上传学校图片
   - **学校简介**（可选）：学校的简要介绍
3. 点击"确认"保存

### 编辑学校

1. 在学校列表中找到目标学校
2. 点击操作列的"编辑"按钮（铅笔图标）
3. 修改需要更新的信息
4. 点击"确认"保存

### 删除学校

> 注意：删除学校会同时删除该学校下的所有班级、教师和学生记录，此操作不可恢复。

1. 在学校列表中找到目标学校
2. 点击操作列的"删除"按钮（垃圾桶图标）
3. 在确认对话框中点击"确认删除"

### 查看学校详情

点击学校名称或"详情"按钮，可查看：
- 学校基本信息
- 校长信息
- 该学校下的班级列表

## 班级管理

### 查看班级列表

1. 点击侧边栏"班级管理"
2. 可通过"学校"下拉框筛选特定学校的班级
3. 使用搜索框按班级名称搜索

### 创建班级

1. 点击"新建班级"按钮
2. 填写班级信息：
   - **班级名称**（必填）：如"一年级一班"
   - **所属学校**（必填）：从下拉列表选择学校
   - **班级图片**（可选）
   - **班级简介**（可选）
3. 点击"确认"保存

### 查看班级详情

点击班级名称进入详情页，可查看：
- 班级基本信息
- 该班级的教师列表
- 该班级的学生列表

---

## 教师管理

### 查看教师列表

1. 点击侧边栏"教师管理"
2. 可按学校或班级筛选
3. 列表显示教师的用户信息（头像、昵称）和所在班级

### 添加教师

1. 点击"添加教师"按钮
2. 选择**班级**（必填）
3. 选择**用户**（必填）：从用户列表中搜索并选择要设为教师的用户
4. 点击"确认"

> 同一用户可以是多个班级的教师。

### 移除教师

1. 在教师列表中找到目标记录
2. 点击"移除"按钮
3. 确认操作

---

## 学生管理

### 查看学生列表

1. 点击侧边栏"学生管理"
2. 可按学校或班级筛选
3. 列表显示学生的用户信息和所在班级

### 添加学生

1. 点击"添加学生"按钮
2. 选择**班级**（必填）
3. 选择**用户**（必填）
4. 点击"确认"

> 注意：一个用户同一时间只能是一个班级的学生。如果该用户已在其他班级，需要先移除再添加。

### 移除学生

1. 在学生列表中找到目标记录
2. 点击"移除"按钮
3. 确认操作

---

## 小组管理

### 查看小组列表

1. 点击侧边栏"小组管理"
2. 列表显示小组名称、描述、成员数量、关联班级数量
3. 使用搜索框按小组名称搜索

### 创建小组

1. 点击"新建小组"按钮
2. 填写小组信息：
   - **小组名称**（必填）
   - **小组描述**（可选）
   - **小组图片**（可选）
   - **详细信息**（可选）
3. 点击"确认"创建

### 查看小组详情

点击小组名称进入详情页，可查看：
- 小组基本信息
- 成员列表
- 关联的班级列表

### 管理小组成员

在小组详情页：

**添加成员**:
1. 点击成员列表右上角"添加成员"
2. 搜索并选择用户
3. 点击"确认"

**移除成员**:
1. 在成员列表中找到目标成员
2. 点击其右侧的"移除"按钮
3. 确认操作

### 关联班级

在小组详情页：

**关联班级**:
1. 点击班级列表右上角"关联班级"
2. 从下拉列表选择班级
3. 点击"确认"

**解除关联**:
1. 在班级列表中找到目标班级
2. 点击"解除关联"按钮
3. 确认操作

## 常见操作

### 搜索和筛选

大多数列表页面支持：
- 顶部搜索框：按名称模糊搜索
- 下拉筛选：按学校、班级等条件筛选
- 分页：使用底部分页控件浏览更多数据

### 图片上传

支持图片上传的表单（学校、班级、小组）：
1. 点击图片区域或"上传图片"按钮
2. 选择本地图片文件（支持 JPG、PNG、GIF，最大 5MB）
3. 图片上传成功后自动显示预览

### 权限说明

不同角色可执行的操作：

| 操作 | 系统管理员 | 校长 | 教师 | 学生 |
|------|-----------|------|------|------|
| 创建/删除学校 | ✓ | - | - | - |
| 编辑学校 | ✓ | ✓（本校）| - | - |
| 创建/删除班级 | ✓ | ✓（本校）| - | - |
| 管理教师/学生 | ✓ | ✓（本校）| - | - |
| 创建/管理小组 | ✓ | ✓ | ✓ | - |
| 查看数据 | ✓ | ✓ | ✓（本班）| ✓（本班）|

## 常见问题

**Q: 点击"学校管理"链接后提示"登录已过期"**

主系统的登录会话已过期，请重新登录主系统后再访问学校管理系统。

**Q: 无法创建学校，提示"无权限"**

创建学校需要系统管理员权限，请联系管理员。

**Q: 添加学生时提示"该用户已在其他班级"**

一个用户同一时间只能是一个班级的学生。请先在原班级移除该学生，再添加到新班级。

**Q: 页面加载缓慢**

可能是网络问题或服务器负载较高，请稍后刷新重试。如果问题持续，请联系系统管理员。
