# 繁体中文支持

## 概述

已为学校管理系统添加繁体中文（zh-TW）语言支持。

## 修改的文件

1. `frontend/src/locales/zh-TW.ts` - 新增繁体中文翻译文件
2. `frontend/src/locales/index.ts` - 添加繁体中文到 i18n 配置
3. `frontend/src/components/common/AppHeader.vue` - 语言切换器添加繁体中文选项

## 支持的语言

- 简体中文 (zh-CN)
- 繁体中文 (zh-TW) ✨ 新增
- English (en-US)

## 使用方法

### 切换语言

用户可以通过页面右上角的语言切换按钮选择语言：
- 简体中文
- 繁體中文
- English

### 默认语言

系统默认使用简体中文，用户选择的语言会保存在 localStorage 中，下次访问时自动应用。

## 开发说明

### 添加新的翻译

如需添加新的翻译文本，需要同步更新三个文件：
1. `zh-CN.ts` - 简体中文
2. `zh-TW.ts` - 繁体中文
3. `en.ts` - 英文

### 翻译规范

1. 保持结构一致：三个语言文件的 key 结构必须完全一致
2. 术语统一：使用标准的繁体中文术语
3. 标点符号：繁体中文使用全形标点符号
4. 专有名词：保持一致的翻译（如"系统管理员"）

## 测试

### 手动测试

1. 启动前端开发服务器
2. 登录系统
3. 点击右上角语言切换按钮
4. 选择"繁體中文"
5. 验证所有页面的文本是否正确显示为繁体中文

### 验证要点

- [ ] 导航菜单显示繁体中文
- [ ] 按钮文本显示繁体中文
- [ ] 表单标签显示繁体中文
- [ ] 提示信息显示繁体中文
- [ ] 错误信息显示繁体中文
- [ ] 确认对话框显示繁体中文

## 注意事项

1. 语言切换后，部分组件可能需要刷新才能完全更新
2. 某些第三方组件（如 Element Plus）的文本可能需要额外配置
3. 日期时间格式可能需要根据语言进行本地化
